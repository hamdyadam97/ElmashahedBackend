# نظام إدارة المعاهد التعليمية

نظام Django MVC متكامل لإدارة المعاهد التعليمية والدبلومات والدورات.

## المميزات

### أنواع الخدمات التعليمية
- **الدبلومات**: مدة الدراسة سنتان، مواعيد بدء وانتهاء مستقلة
- **الدورات**: مدة الدراسة 6 أشهر، نظام تسجيل مستقل

### أنواع المستخدمين والصلاحيات

1. **Admin / CEO**
   - صلاحيات كاملة على النظام
   - رؤية جميع الفروع والتقارير
   - إدارة المستخدمين والصلاحيات

2. **المدير الإقليمي**
   - إشراف على مجموعة من المعاهد
   - تقارير الأداء للفروع التابعة

3. **مدير الفرع**
   - مسؤول عن فرع واحد
   - رؤية جميع العمليات داخل الفرع
   - تقارير شهرية للفرع

4. **الموظف**
   - تسجيل العملاء
   - إصدار الأذونات
   - مشاهدة التسجيلات الخاصة به فقط

5. **العملاء**
   - لا يملكون حساب دخول
   - يتم تسجيل بياناتهم من قبل الموظفين

### نظام الأذونات (PDF)
- توليد PDF مخصص لكل معهد
- يحتوي على شعار المعهد، اسم المعهد، رقم الترخيص
- بيانات الطالب ونوع التسجيل ومواعيد الدراسة

### التقارير
- تقارير الموظفين (شهرية)
- تقارير مديري الفروع
- تقارير المديرين الإقليميين
- تقارير شاملة للأدمن

## التثبيت

1.克隆 المستودع
```bash
git clone <repository-url>
cd django_edu_system
```

2. إنشاء بيئة افتراضية
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate  # Windows
```

3. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

4. إنشاء قاعدة البيانات
```bash
python manage.py migrate
```

5. إنشاء مستخدم أدمن
```bash
python manage.py createsuperuser
```

6. تشغيل الخادم
```bash
python manage.py runserver
```

## الاستخدام

1. افتح المتصفح على `http://localhost:8000`
2. سجل الدخول باستخدام بيانات الأدمن
3. قم بإنشاء المعاهد والمستخدمين
4. أضف الدبلومات والدورات
5. سجل العملاء وأصدر الأذونات
