# دليل البدء السريع

## تشغيل النظام

### 1. تثبيت المتطلبات
```bash
pip install django pillow reportlab
```

### 2. تشغيل الخادم
```bash
python3 manage.py runserver
```

أو استخدم السكربت الجاهز:
```bash
bash start.sh
```

### 3. فتح المتصفح
افتح المتصفح على العنوان: `http://localhost:8000`

## الهيكل التنظيمي

```
النظام
├── المعاهد (Institutes)
│   ├── معهد التقنية العالي (مثال)
│   └── ...
├── المستخدمين
│   ├── Admin (صلاحيات كاملة)
│   ├── مدير إقليمي (عدة معاهد)
│   ├── مدير فرع (معهد واحد)
│   └── موظف (معهد واحد - صلاحيات محدودة)
├── البرامج
│   ├── دبلومات (24 شهر)
│   └── دورات (6 أشهر)
├── العملاء
│   └── بيانات شخصية + تسجيلات
└── الأذونات
    └── PDF مخصص لكل معهد
```

## الخطوات الأساسية

### 1. إنشاء معهد جديد (Admin)
- اذهب إلى "المعاهد"
- اضغط "إضافة معهد"
- أدخل البيانات المطلوبة
- ارفع الشعار والتوقيع

### 2. إضافة مستخدمين (Admin)
- اذهب إلى "المستخدمين"
- اضغط "إضافة مستخدم"
- حدد الدور والمعهد

### 3. إضافة برامج (مدير فرع)
- اذهب إلى "البرامج" → "الدبلومات" أو "الدورات"
- اضغط "إضافة"
- حدد المواعيد والتكلفة

### 4. تسجيل عميل (موظف)
- اذهب إلى "العملاء"
- اضغط "تسجيل عميل"
- أدخل البيانات الشخصية

### 5. إصدار إذن (موظف)
- اذهب إلى "الأذونات"
- اضغط "إصدار إذن"
- اختر العميل والبرنامج
- اضغط "إصدار الإذن"
- سيتم إنشاء PDF تلقائياً

## التقارير

- **الموظف**: تقريره الشخصي فقط
- **مدير الفرع**: تقرير الفرع + أداء الموظفين
- **المدير الإقليمي**: تقارير مجمعة للفروع
- **Admin**: تقارير شاملة لكل النظام

## ملاحظات هامة

1. كل معهد له قالب PDF مخصص
2. الموظف يرى بيانات معهده فقط
3. الأذونات تولد PDF تلقائياً
4. العملاء لا يملكون حسابات دخول
